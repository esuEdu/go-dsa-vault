# go-dsa-vault

Personal study vault for Data Structures & Algorithms using Go. This repo doubles as an Obsidian vault — open the root folder in Obsidian to get backlinks, tags, and graph view across notes and problem write-ups.

---

## Structure

```
go-dsa-vault/
├── concepts/           # Theory notes (Big O, Go syntax, patterns)
├── data-structures/    # Implementations built from scratch
│   ├── array/
│   ├── linked-list/
│   ├── stack/
│   ├── queue/
│   ├── hash-map/
│   ├── tree/
│   ├── heap/
│   ├── graph/
│   └── trie/
├── algorithms/         # Algorithm patterns
│   ├── sorting/
│   ├── searching/
│   ├── binary-search/
│   ├── two-pointers/
│   ├── sliding-window/
│   ├── dynamic-programming/
│   └── backtracking/
└── leetcode/           # LeetCode problems by difficulty
    ├── easy/
    ├── medium/
    └── hard/
```
